# SOP 目录

此目录用于存放经过验证的"正确路径"标准操作流程（SOP）。

## 用途

Worker 在执行任务前会检查此目录，如果有与当前任务相关的 SOP，会优先遵循 SOP 中的步骤。

## 文件命名规范

- 使用小写字母和下划线
- 描述任务类型，如：`code_analysis.md`、`git_clone.md`、`dependency_check.md`

## SOP 文件格式

```markdown
# [任务类型名称]

## 适用场景
[描述什么情况下使用此 SOP]

## 标准步骤
1. [步骤 1]
2. [步骤 2]
3. ...

## 注意事项
- [注意点 1]
- [注意点 2]
```

## 如何贡献

Worker 在执行任务时如果发现可复用的正确路径，会在回答末尾建议 SOP。主控审核后决定是否采纳并写入此目录。
